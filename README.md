# 项目简介
猛哥最近越来越内向，聊微信也不喜欢发语音，就想所有说的话，都通过打字来说，从而有了这个项目
# 技术堆栈
* 服务端框架是c# core webapi
* 前端目前是vue +jquery 
* UI主要是zmUI
* 服务端核心功能是阿里云语音服务（讯飞api示例代码太少了）
# 阶段开发计划
1. 了解阿里云语音服务(完成)
2. 输入文字，返回语音(完成)【竟然自动识别中英文，为阿里点个赞】
3. 客户端播放语音(完成)
4. 保存语音(完成)，实现快捷模块
5. 自定义保存快捷语句(完成)
